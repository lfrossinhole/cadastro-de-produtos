# Sistema de Cadastro de Produtos

> Status do projeto: Em desenvolvimento

Para rodar essse projeto no seu computador, por favor digite no terminal:

```
npm install react
```

Agora eu vou escrever um código em HTML para testar:

```
<!doctype html>
<html>
  <head>
      <meta charset="utf-8"/>
      <title>Sistema</title>
  </head>
  <body>
    <p>Olá mundo!</p>
  </body>
</html>
```
